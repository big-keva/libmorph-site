Загрузки

Для каждого языка доступны два типа модулей: словарные (libmorph/*) для точного анализа известных слов и вероятностные (libfuzzy/*) для обработки неизвестных слов. Модули в разработке отмечены как неактивные.

Обертки для языков

Язык Пакет Команда установки
Python libmorphrus pip install libmorphrus
Статусы модулей:
готов — стабильная версия, рекомендуется для производства
бета — тестирование, возможны изменения API
разработка — в активной разработке, не для производства
Для Windows: Требуется Visual C++ Redistributable 2019+. Вероятностные модули требуют больше оперативной памяти (от 500 MB).
