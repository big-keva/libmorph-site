📦 Установка из репозитория (Debian/Ubuntu)

Подключение репозитория:

curl -sSf https://libmorph.ru/deb/public.key | gpg --dearmor | sudo tee /usr/share/keyrings/libmorph-archive-keyring.gpg
echo "deb https://libmorph.ru/deb stable main" | sudo tee /etc/apt/sources.list.d/libmorph.list
apt update

Пакеты:

apt install libmorph-rus
apt install libmorph-rus-dev
