Модули libmorph

Для каждого языка доступны два типа модулей: словарные (libmorph/*) для точного анализа известных слов и вероятностные (libfuzzy/*) для обработки неизвестных слов, опечаток, неологизмов. Некоторые модули находятся в разработке и будут доступны в ближайших релизах.

🇷🇺 Русский язык

libmorph/rus словарный

готов точный анализ

Полная морфология русского языка на основе словарных данных. Высокая точность для известных слов.

  • ✅ Объём словаря: 2.5 млн словоформ
  • ✅ Поддержка неологизмов (2000+ слов)
  • ✅ Склонение фамилий и имён
  • ✅ Распознавание аббревиатур
  • ✅ Грамматические категории: падеж, число, род, одушевлённость
Инициализация:
IMlmaMb* morph = CreateRussianMorphMb("ru_RU.UTF-8");
IMlmaWc* morph_w = CreateRussianMorphWc();

libfuzzy/rus вероятностный

разработка обработка неизвестных

Вероятностный анализатор для обработки неизвестных слов, опечаток и неологизмов на основе статистических моделей.

  • ⏳ Модель на основе FastText (500K векторов)
  • ⏳ Предсказание грамматических форм для неизвестных слов
  • ⏳ Исправление опечаток (Levenshtein distance)
  • ⏳ Вероятностная лемматизация
Планируемый API:
IFuzzyMb* fuzzy = CreateRussianFuzzyMb("ru_RU");
fuzzy->GuessLemma("програмирование");
Ожидайте в версии 2.5.0

libfuzzy/eng вероятностный

бета обработка неизвестных

Вероятностный анализатор для английского языка. Доступен в бета-версии для тестирования.

  • 🔄 Распознавание неологизмов

libfuzzy/ukr вероятностный

разработка обробка невідомих

Імовірнісний аналізатор для обробки невідомих слів, діалектизмів та неологізмів.

  • ⏳ Модель на основі FastText

  • ⏳ Передбачення граматичних форм
Планується:

Очікуйте у версії 2.6.0

Сравнение типов модулей

Характеристика Словарный (libmorph/*) Вероятностный (libfuzzy/*)
Точность на известных словах 99.9% 85-95% (зависит от модели)
Обработка неизвестных слов ❌ Не работает ✅ Да (статистическое предсказание)
Исправление опечаток ❌ Нет ✅ Да (Levenshtein, Damerau)
Скорость (слов/сек) 1 000 000+ 50 000 - 100 000
Память 50-100 MB 200-500 MB (модель)
Рекомендация Для известной лексики, высокая точность Для текстов с опечатками, новых слов
Рекомендации по использованию:
Для максимальной точности рекомендуется комбинировать оба подхода: сначала пробовать словарный анализ, при неудаче — вероятностный. В будущих версиях появится гибридный модуль, объединяющий оба метода.
