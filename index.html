libmorph: морфологический анализ на C++

1 000 000 слов/сек на одном ядре · в 50+ раз быстрее pymorphy3

libmorph — это семейство высокопроизводительных библиотек морфологического анализа, написанных на C++. Они предоставляют единый COM-подобный API для работы с русским, английским и украинским языками.

Ключевые особенности

  • Скорость: анализ 1 миллиона слов в секунду на одном ядре CPU
  • Две версии API: multibyte (с поддержкой кодовых страниц) и widechar (UTF-16)
  • Обертки: PHP, Python, Golang и другие языки
  • Лемматизация, проверка орфографии, построение словоформ
  • Готовые пакеты: .deb, .rpm, ZIP для Windows

Пример использования (C++)

// Лемматизация слова (multibyte-версия)
#include <libmorph/rus.h>

IMlmaMb* pMorph = CreateRusMorph();
if (pMorph) {
    char lemma[256];
    pMorph->Lemmatize("бежали", 6, 
                     nullptr, 0, lemma, 256, 
                     nullptr, 0, 0);
    // lemma = "бежать"
    pMorph->Detach();
}
